# Sketch Manifest

## Design Direction
Dark-native UI matching the FlowSpace app palette (surface-0 through surface-4, accent #7c6af7). Interactions should feel fast and minimal — AI features integrate into the existing workflow without adding visual noise.

## Reference Points
- Existing AiTextToolbar in BlockEditor (page editor)
- Existing AiPanel floating chat (board-level)
- FlowSpace board card design language
