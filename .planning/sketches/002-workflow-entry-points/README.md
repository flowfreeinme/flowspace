# Sketch 002: Workflow Entry Points

## Design Question
Where and how do users access Kanban, Flowchart, and Timeline blocks on the board?

## How to View
open .planning/sketches/002-workflow-entry-points/index.html

## Variants
- **A: Toolbox submenu + right-click section** — Workflow button in toolbox opens a submenu; right-click menu also exposes all three options under a "Workflow" section label
- **B: Separate toolbox buttons** — Three dedicated buttons (Kanban, Flowchart, Timeline) in the toolbox with an accent color grouping
- **C: Right-click flyout** — Workflow is a hoverable flyout in the right-click menu; toolbox stays minimal

## Winner
User confirmed: right-click context menu is the primary creation point. Variant A (right-click section + toolbox submenu) balances discoverability with toolbox space.

## What to Look For
- Does the toolbox get too tall with workflow buttons?
- Is the right-click "Workflow" section label clear enough?
- Flyout (C) vs flat list (A) — which is faster to reach?
