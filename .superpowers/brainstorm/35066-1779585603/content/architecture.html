HomeScreen props passed to each widget

config: WidgetConfigMap[type] // settings from getWidgetSettings()
That's it. Each widget reads its own stores directly.

One special case: CalendarWidget navigation

Clicking a day in the calendar replaces the entire HomeScreen with DayView / WeekView (fullscreen). CalendarWidget calls two callbacks that HomeScreen handles:

onSelectDay(date: Date) // HomeScreen renders DayView
onSelectWeek(date: Date) // HomeScreen renders WeekView
